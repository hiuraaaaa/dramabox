Search Drama

Cari drama berdasarkan keyword dengan filter sort dan pagination otomatis.

GET /api/search Search by keyword ▼
Parameters
q *
string — keyword pencarian
sort
1=Trending 2=Latest 3=Unwatched
GET /api/search?q=&sort=1
